# GEN AI Systems Paper Reading List

This repository contains curated reading lists and notes for research papers in
**AI systems, distributed systems, and large-scale ML infrastructure**.

The goal of this repo is to organize papers thematically, preserve useful links,
and support structured reading and critique over time.

Each folder corresponds to a high-level topic, and contains a single `paper_name.md`
file with links to papers, videos, slides, and other relevant material.

## `paper_name.md` Format
